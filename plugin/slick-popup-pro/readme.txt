A lightweight WordPress Popup plugin that converts any Contact Form 7 form into a customizable pop-up form which is slick, beautiful and responsive to different screen-sizes. You can create as many popups and place them on any page using a shortcode. 

== Description ==

Slick Popup Pro is a lightweight Contact Form 7 Popup Plugin.

It uses the Conact Form 7 form and converts it into a responsive popup, which is Slick, Beautiful and perfectly responsive. It quickly give you a pop-up form that aids your visitors to contact you easily using an enquiry form, contact form or a quote form. Enjoy with the plugin. 

The plugin converts any Contact Form 7 form in to a beautiful popup. So, Contact Form 7 plugin must be installed and setup to use Slick Popup Pro. 

*   Compatible with Contact Form 7
*   Responsive and browser compatible
* 	Lightweight and fast
*	Pre-built and custom color schemes
*	Easy to customize

[Plugin Demo](http://www.slickpopup.com/)

The new features in the pro version are just awesome. You will find it easier to customize and get your desired form in a popup. Be it an enquiry form, contact form quote form or any other.

== Installation ==

The quickest method for installing the Slick Popup Pro plugin is:

* Login to your WordPress account
* Go to Plugins -> Add New
* Click on Upload Link
* Browse to slick-popup-pro.zip file (it is located in the folder you have downloaded from Codecanyon) and choose that file
* Click “Install Now” button
* Wait while plugin is uploading to your server
* Click “Activate Plugin” button
* Now you are ready to Configure your Slick Popup

Alternatively, you can you just extract the zip file and upload the folder 'slick-popup-pro' using FTP.
If you would really prefer to do things manually then follow these instructions:

1. Extract the zip file: 'slick-popup-pro.zip'.
2. Upload the `slick-popup-pro` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure settings from Dashboard Options page "Slick Popup Pro".
